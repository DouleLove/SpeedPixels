<br><br>

## Installing
**Python 3.10+ is required**<br><br>
To clone repository and install required packages use the following command:<br>
<pre>
$ git clone https://github.com/DouleLove/SpeedPixels.git
$ cd SpeedPixels
$ python -m pip install -r requirements.txt
</pre>
